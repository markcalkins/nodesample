OpenWest Node/Express Project - Getting Started

Thank you for downloading this project. The project is self documenting, but you need enough information to get started, which is the purpose of this readme.

Do This First

There are a few things you will need to get started:

  1. If you do not have Node.js on your computer, you'll need to download and install it. You can do that by going to nodejs.org.
  2. You will also need Bower to install a few other components for your app. Once you have Node.js installed, it is pretty easy:

    3. npm install -g bower

  3. The application uses MongoDB for storing blogs. You'll need to install MongoDB. This page shows you how.
  4. Create a new directory for your project and move all the downloaded project files to that directory.
  5. Now you can install all of the node modules and bower components. You can by running these commands in your terminal in the same directory where you moved your files.

    6. npm install
    bower install

Now You Can Run The App

Once you have all the modules installed, you can run the app:

  1. In terminal (located at the directory where you moved the project files), you can start the local MongoDB database (local to this application):

    2. mongod --dbpath ./lib/db

  2. Open a second terminal window in the same directory.
  3. Launch the Node app:

    4. node app.js

  4. Click on this link to open a browser window to show the app running.

You are now up and running!

Note the blog mgr linka> in the footer. That will take you to the blog manager where you can create new or edit existing blogs. You'll need to create a new user and password first, then log into the blog manager.
